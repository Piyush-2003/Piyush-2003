I'm a Data Science student with hands-on experience in the hospitality industry, which means most of my projects use real business data — not cleaned-up textbook datasets. ## What I've been working on

**Goomti Restaurant Analytics Portfolio** — Five end-to-end data projects built on real F&B sales data from an Indian restaurant: Power BI dashboard, menu engineering (Boston Matrix), K-Means customer segmentation, revenue forecasting, and a live Streamlit pricing app. Then rebuilt as four React applications — all deployed live on Vercel.

**CarePoint365 React Apps** — Three React applications built to mirror the modules of a UK care management software platform: a Time & Attendance Monitor with a live real-time clock, a Recruitment Pipeline Kanban board with compliance tracking, and a Care Home Management Dashboard covering residents, incidents and compliance. All deployed live.

**Deep Learning** — Fine-grained bird species classification using ResNet transfer learning on CUB-200-2011, and a character-level LSTM trained on Agatha Christie's Poirot stories with temperature sampling at inference time.

**Advanced AI** — Food image segmentation benchmarking U-Net vs DeepLabV3+ on FoodSeg103, and multimodal idiom representation comparing CLIP and BERT embeddings with UMAP visualisation and XGBoost classification.

**Machine Learning** — Injury severity prediction on 100K+ US road accident records (FARS). Class imbalance handled with SMOTE, hyperparameter tuning with HalvingRandomSearchCV, evaluated on Macro F1 rather than accuracy.

**MSc Dissertation (Ongoing)** — Interpretability of deep neural network representations on MNIST using t-SNE, UMAP, SHAP DeepExplainer, Vanilla Saliency, SmoothGrad and Integrated Gradients.

**Generative AI** — A storytelling app for children built with the OpenAI API and text-to-speech output, plus a bias detection pipeline surfacing skewed gender and thematic representation in NLP datasets.

**Image Processing** — Medical image analysis on OCT retinal scans using classical techniques: Otsu thresholding, Sobel edge detection, Gaussian filtering.
